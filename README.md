# App ativiade de teste  ideafix

# Iniciando a API e o APP

``` bash
# Express API
cd api

# Instale dependências
npm install

# Inicia Express
npm start

# React APP
cd app

# Instale dependências
npm install

# Inicia o app
npm start

# O Server Express inicia no link http://localhost:5000 o app no link http://localhost:3000
```
